# Multi-Agent Business Operations Assistant
## Live Demo

https://multi-agent-business-assistant-jjwmpgrmtqypuvmr2yv7fj.streamlit.app/

## Overview

A Multi-Agent AI platform built using Python, Streamlit, Gemini AI, Pandas, and SQLite.

The system automates business operations through specialized AI agents and provides business insights, task prioritization, email analysis, meeting summarization, and natural language SQL querying.

## Features

### Email Agent

* Email priority detection
* Email summarization
* Suggested replies

### Meeting Agent

* Meeting notes summarization
* Action item extraction

### Task Agent

* Task prioritization
* High/Medium/Low classification

### Report Agent

* CSV/Excel upload
* Business insights generation
* Trend analysis
* Recommendations

### Manager Agent

* Executive summary generation
* Risk analysis
* Decision recommendations

### SQL Agent

* Natural language to SQL conversion
* SQLite database querying
* Automated business analytics

## Technologies Used
